# Overview
// Is an outdoor training app. That provides the user with outdoor training places throughout Stockholm. 
// Connected with my own small backend and mongoDb

## Run 
//  1 - npm use node, 
//  2 - Install dependencies - npm install
//  3 - npm start 

## Technologies
- MongoDB
- React Native 
- Node.js
